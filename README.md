# capstone-recipe-finder
Project name - Recipe Finder

Project Overview
Recipe Finder is a web application that allows users to search for recipes by ingredients or dish names. Users can view recipe details, including ingredients, preparation instructions, and a demonstration video. This project is built with React and styled using Tailwind CSS.

Features
Search Recipes: Search for recipes by dish name or ingredients using TheMealDB API.
Recipe Details: View detailed information for each recipe, including:
Recipe image
Ingredients list with quantities
Preparation instructions
Embedded YouTube video 
Responsive Design: Optimized for desktop, tablet, and mobile using Tailwind CSS.

Technologies Used
Frontend: React, JavaScript, Tailwind CSS
API: TheMealDB API
Deployment: Netlify or Vercel
Version Control: Git and GitHub
